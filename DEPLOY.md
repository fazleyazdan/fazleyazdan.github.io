# 🚀 Portfolio Deployment Guide
## Deploy for FREE using GitHub Pages

---

## ✅ Step 1 — Personalize Your Portfolio (Do this FIRST!)

Open `index.html` and replace these placeholders:

| Placeholder | Replace With |
|---|---|
| `your.email@example.com` | Your actual email |
| `YOUR_USERNAME` (in GitHub links) | Your GitHub username |
| `YOUR_USERNAME` (in LinkedIn) | Your LinkedIn profile slug |

---

## ✅ Step 2 — Create a GitHub Account (if you don't have one)

1. Go to **https://github.com**
2. Click **Sign Up** — it's completely free
3. Choose a username (ideally your name or professional handle)

---

## ✅ Step 3 — Create a New Repository

1. Once logged in, click the **+** icon (top right) → **New repository**
2. Name it exactly: `YOUR_USERNAME.github.io`
   - Example: if your username is `fazleyazdanqa`, name it `fazleyazdanqa.github.io`
3. Set it to **Public**
4. ✅ Check "Add a README file"
5. Click **Create repository**

---

## ✅ Step 4 — Upload Your Portfolio Files

1. In your new repository, click **Add file** → **Upload files**
2. Drag and drop these 3 files from `d:\My Projects\my-portfolio\`:
   - `index.html`
   - `style.css`
   - `script.js`
3. Scroll down, click **Commit changes**

---

## ✅ Step 5 — Enable GitHub Pages

1. In your repository, click **Settings** (top tab)
2. Scroll down to **Pages** (left sidebar)
3. Under **Source**, select **Deploy from a branch**
4. Branch: **main** | Folder: **/ (root)**
5. Click **Save**

---

## 🎉 Step 6 — Your Site is LIVE!

Wait **1–2 minutes**, then visit:
```
https://YOUR_USERNAME.github.io
```

That's your **free, permanent portfolio URL** you can share on LinkedIn, CVs, etc!

---

## 🔄 How to Update in the Future

Whenever you want to make changes:
1. Edit the files on your computer
2. Go to your GitHub repo
3. Click the file → pencil icon ✏️ to edit
   OR upload a new version via **Add file → Upload files**
4. Commit → site auto-updates in ~30 seconds

---

## 📝 What to Update in index.html

Search (`Ctrl+F`) for these strings and replace:

```
your.email@example.com       → your real email
YOUR_USERNAME                → your GitHub username
```

Also update the `<title>` and `<meta name="description">` if you want
to change your displayed name from "fazleyazdan".

---

## 💡 Tips

- Your URL will be `https://YOUR_USERNAME.github.io` — share it everywhere!
- GitHub Pages is **100% free** with no limits for static sites
- The site is HTTPS-secured automatically by GitHub
- No domain purchase needed — ever!
